Politique de confidentialité (RGPD)

En résumé

Le site ne pose aucun cookie, ne vous suit pas à travers d'autres sites, ne partage rien avec des régies publicitaires, et n'utilise aucun traceur tiers. Aucune bannière de consentement n'est nécessaire car aucun traitement soumis à consentement n'est effectué.

Responsable du traitement

Statistiques anonymes de fréquentation (GoatCounter)

J'utilise GoatCounter pour mesurer l'audience du site. Cet outil est conçu pour respecter la vie privée :

  • Aucun cookie déposé sur votre navigateur
  • Votre adresse IP est hachée avec un sel qui change chaque jour ; elle n'est jamais stockée
  • Aucun identifiant individuel, aucun profil utilisateur
  • Les données sont agrégées : pages vues, pays, navigateur, site référent
  • Hébergement des statistiques en Europe (Pays-Bas)

Selon les lignes directrices de la CNIL, ce type de mesure d'audience anonyme est exempté de consentement car strictement nécessaire à l'exploitation du service et sans portée publicitaire.

Commentaires (Giscus / GitHub Discussions)

Les commentaires sont gérés par Giscus, qui s'appuie sur les GitHub Discussions du dépôt public du site. Pour laisser un commentaire :

  • Vous devez disposer d'un compte GitHub et y consentir en vous connectant
  • Votre pseudonyme et avatar GitHub s'affichent publiquement à côté du commentaire
  • Le contenu du commentaire est stocké sur GitHub ; il est soumis à la politique de confidentialité de GitHub
  • Le chargement de Giscus est différé : il ne s'effectue que lorsque vous faites défiler jusqu'à la zone de commentaires

Préférence de thème (localStorage)

Si vous cliquez sur le bouton de changement de thème (clair / sombre) dans la navbar, votre choix est mémorisé dans le localStorage de votre navigateur (clé theme, valeurs light ou dark). Il s'agit d'un stockage purement local, jamais transmis au serveur, utilisé uniquement pour vous servir le thème que vous avez explicitement choisi.

Selon les lignes directrices de la CNIL, ce stockage est exempté de consentement car strictement nécessaire à la fourniture d'un service explicitement demandé par l'utilisateur (préférence d'affichage). Si vous n'utilisez pas le bouton, aucune valeur n'est écrite : le site se contente alors de respecter la préférence de votre système d'exploitation (prefers-color-scheme).

Vous pouvez effacer cette préférence à tout moment depuis les paramètres de votre navigateur (suppression du stockage de site).

Aucune autre collecte

Le site ne collecte rien d'autre :

  • Pas de formulaire de contact ou d'inscription
  • Pas de newsletter
  • Pas d'intégration de réseaux sociaux traquant les visiteurs (pas de bouton "J'aime", de widget Twitter, etc.) — les boutons de partage présents en bas des articles sont de simples liens HTML vers les pages d'intention de publication des plateformes (LinkedIn, Bluesky, X) : aucun script tiers chargé, aucune donnée transmise tant que vous ne cliquez pas
  • Toutes les ressources (polices, scripts, images, bibliothèques) sont hébergées sur le même domaine : aucune requête vers des CDN tiers
