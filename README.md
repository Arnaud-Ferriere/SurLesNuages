# Sur les nuages

Site personnel d'**Arnaud Ferriere** — articles techniques sur l'administration
système, Microsoft 365, la gestion des identités hybrides (AD / Entra ID) et
l'automatisation (PowerShell, Power Platform).

🔗 **Site live** : <https://surlesnuages.fr>

## Stack

Jekyll + GitHub Pages, Bootstrap 5.3, Font Awesome 6.7, Mermaid, Giscus.

Basé sur le template open source [Cirrus for Jekyll](https://github.com/Arnaud-Ferriere/Cirrus-for-Jekyll)
(CC BY 4.0).

## Build local

```bash
bundle install
bundle exec jekyll serve --livereload --drafts
```

Une coquille, un lien cassé, une info périmée ? Ouvrir une [issue](https://github.com/Arnaud-Ferriere/SurLesNuages/issues)

qui pointe vers la source GitHub directement.
