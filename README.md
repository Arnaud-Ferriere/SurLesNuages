# Sur les nuages

Mon site personnel, articles techniques sur l'administration
système, Microsoft 365, la gestion des identités hybrides (AD / Entra ID) et
l'automatisation (PowerShell, Power Platform).

**Site live** : <https://surlesnuages.fr>

## Stack
Jekyll + GitHub Pages, Bootstrap 5.3, Font Awesome 6.7, Mermaid, Giscus.
Basé sur mon template open source [Cirrus for Jekyll](https://github.com/Arnaud-Ferriere/Cirrus-for-Jekyll)
(CC BY 4.0).
Détails et licences des bibliothèques tierces sur [/conformite/](https://surlesnuages.fr/conformite/).

## Signaler une erreur

Une coquille, un lien cassé, une info périmée ou même une explication pas claire ? N'hésitez pas à ouvrir une [issue](https://github.com/Arnaud-Ferriere/SurLesNuages/issues) ou une PR. Chaque article expose un bouton **`.md`** dans son entête
qui pointe vers la source GitHub directement.
