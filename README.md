# Sur les nuages

Mon site personnel, j'y rédige des articles techniques sur l'administration
système, Microsoft 365, la gestion des identités hybrides (AD / Entra ID) et
l'automatisation (PowerShell, Power Platform).

**Site live** : <https://surlesnuages.fr>

Les scripts réalisés pour les articles sont hébergés sur le Repo dédié [Toolkit SurLesNuages](https://github.com/Arnaud-Ferriere/toolkit-surlesnuages/). J'y publie aussi une "Boite à outils" de scripts et modules PowerShell pour l'administration et l'automatisation Active Directory, Microsoft 365, Entra ID, Azure etc
## Stack
Jekyll + GitHub Pages, Bootstrap 5.3, Font Awesome 6.7, Mermaid 11, Giscus.
Basé sur mon template open source [Cirrus for Jekyll](https://github.com/Arnaud-Ferriere/Cirrus-for-Jekyll)
(CC BY 4.0).
Détails et licences des bibliothèques tierces sur [/conformite/](https://surlesnuages.fr/conformite/).

## Signaler une erreur

Une coquille, une info périmée ou même une explication pas claire ? Je prends le temps de faire des recherches pour que les informations publiées soient dans l'état de l'art, mais je ne suis pas à l'abri d'une erreur ! N'hésitez pas à ouvrir une [issue](https://github.com/Arnaud-Ferriere/SurLesNuages/issues) ou une PR. Chaque article expose un bouton **`.md`** dans son entête
qui pointe vers la source GitHub directement.
